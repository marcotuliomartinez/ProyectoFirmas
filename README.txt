# ProyectoFirmas
Este repositorio contiene el código fuente para cifrar y descifrar archivos de texto con el algoritmo de cifrado Cesar y el generador de contraseñas Diceware
Para poder utilizar la  aplicacion se debe cambiar la extencion del archivo ProyectoFinalFirmas.txt por .java
para compilra  la aplicacion desde linux utilize  el comando javac  -encoding ISO-8859-1 ProyectoFinalFirmas.java
